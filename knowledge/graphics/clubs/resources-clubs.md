# Club resources

> *"Read `resources-clubs.md` as your primary design source for clubs only. Pay special attention to the 'Keywords' and 'Suggested Theme' to ensure the asset matches the {user}'s request."*
---
> Definition: clubs are images which can be used, optional use, as micro headers for specific sections and columns. Clubs on various designs can be 1, 2, 3 or more columns listing premium users. Clubs can be named as Premiu Tippers, Vips, so on.
Name of the clubs can differ from one column or Tier to the other. Common names can be: Silver, Gold, Diamnon tiers or other similar names which define an encrease of value.
